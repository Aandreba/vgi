# Entorn VGI

## System dependencies

This are the dependencies you'll have to manually install on your system to build and compile.

- Vulkan SDK (1.3 or higher)
- CMake (3.4 or higher)
- Just (optional)
- Doxygen (optional)
- GraphViz (optional)

## Tested compilers

Other compilers **should** work, but have not been tested.

- Visual Studio 17 2022
- Apple Clang 17.0
